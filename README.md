# finly.

A modern personal finance tracker built using React + Vite.

## Features

- Add income & expense transactions
- Dashboard with financial summaries
- Monthly analytics charts
- Category-wise budget tracking
- Transaction filtering & sorting
- Persistent storage using localStorage
- Responsive UI

## Tech Stack

- React
- Vite
- React Router DOM
- Recharts
- CSS
- localStorage

## Screens

- Login
- Dashboard
- Transactions
- Budget
- Analytics

## Installation

```bash
npm install
npm run dev
```
